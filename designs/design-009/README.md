# Design 009 - Playground Mission Control

This exploration treats WordPress Playground as a three-pane operations console:

- Left: launch flows, blueprint shortcuts, and saved Playgrounds.
- Center: the active WordPress browser frame with path navigation and refresh.
- Right: Site Manager tools for settings, files, blueprints, database, and logs.

Secondary workflows use a contextual drawer so creation, saving, saved management, settings, export/import, PR starts, GitHub import, and the Blueprint gallery remain connected to the active site instead of replacing it.
