# Design 082 - Saved Sites Library

This exploration makes saved and temporary Playgrounds the first-class workspace. Creation, import, saving, reset, export, and Site Manager tools are framed as actions on the selected library item, with the live WordPress preview intentionally secondary.

The layout uses dense IDE-like panels on desktop and a bottom navigation model on mobile so the library, start routes, selected site, and tools remain separate task surfaces instead of collapsing into a long desktop stack.
