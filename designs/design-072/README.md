# Design 072 - Playground Lesson Path Workspace

This static redesign frames WordPress Playground as a modal-free onboarding checklist for first-time WordPress learners. The first viewport is an actual app workspace with the live site preview, current temporary state, and a sequential path through launch, save, saved management, Site Manager tools, Blueprint gallery, transfer actions, and final WordPress result.

Open `index.html` directly in a browser or through GitHub Pages. No build step is required.
