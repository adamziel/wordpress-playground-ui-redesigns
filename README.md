# WordPress Playground UI Redesigns

One hundred static, high-fidelity redesign explorations for the WordPress Playground UI.

Each design preserves the current Playground product goals and feature surface captured in `research/PLAYGROUND_UI_MAP.md`: creating Playgrounds, saving and managing them, switching settings, navigating the embedded WordPress site, managing files, running blueprints, inspecting database/log state, exporting, importing, and starting from existing templates or PR flows.

The gallery is published from `index.html` and each exploration lives in `designs/design-###/`.
