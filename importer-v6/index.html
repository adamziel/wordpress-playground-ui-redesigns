Universal WordPress Importer · v6

Thirty design explorations for the Importer admin UI

Six themes × four variations each, plus a follow-up Memo Column refinement set (25–30) that sharpens how the user picks between pasting a URL and uploading a file or folder — with progressively quieter drop-zone chrome. Every design demonstrates the same end-to-end import script (Source → Configure → Confirm → Run → Done).
