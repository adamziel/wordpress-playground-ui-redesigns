# Universal WordPress Importer — v6 design gallery

24 distinct design explorations for the Universal WordPress Importer admin UI redesign. Browse them all side by side at [`index.html`](index.html), or open any single design under [`options/`](options/).

## What this is

Six themes &times; four variations each, all rendering the same end-to-end import script (Source &rarr; Configure &rarr; Confirm &rarr; Run &rarr; Done) so chrome, density, and decision-point handling can be compared directly.

## The six themes

| Theme | Designs | Motivation |
| --- | --- | --- |
| **Quiet Modal** | 01–04 | Self-contained modal over a faint admin backdrop; the lightest footprint. |
| **Page Canvas** | 05–08 | Full-page WP-admin surface; the importer reads as a document, not a popup. |
| **Right Sheet** | 09–12 | Slide-in side sheet; lets the admin stay visible behind. |
| **Conversation** | 13–16 | Transcript-shaped flow; each stage is a memo or turn, not a panel. |
| **Cockpit Mini** | 17–20 | Dense, ops-tinted console; keyboard-first, telemetry-friendly. |
| **Split View** | 21–24 | Source and outcome pinned as parallel surfaces; the import is a transformation. |

## How to read it

Each design demonstrates the same demo script end to end. The gallery is a static HTML page: every preview is rendered in its own iframe, so the embed and the full page behave identically. Click "Open full page &rarr;" beneath any preview to view a design at native width in a new tab.

Designs use system fonts and no external resources — they're safe to view offline.
